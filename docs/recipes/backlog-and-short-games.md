# Backlog and short games

## Backlog candidates

Combine a personal status or local tag with objective exclusions. For example, exclude games already marked `playing` or `completed`, then assign the remaining candidates to a backlog collection.

## Short games

Prepare HLTB data, then start with:

```text
hltb:<20
```

Add other filters to narrow the result, such as a platform, genre, or minimum review score. Preview missing HLTB entries separately so they are not silently treated as long games.

## Save as separate intent

Use a name such as `Backlog - Under 20 hours`. That preserves the distinction between a broad backlog and a shorter next-play queue.
