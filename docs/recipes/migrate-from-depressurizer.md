# Migrate from Depressurizer

Repressurizer is a separate application inspired by Depressurizer. It does not replace the original program in place.

## Recommended migration

1. Keep the original Depressurizer database and Steam collection files backed up.
2. Load Steam's current collections into Repressurizer.
3. Import the Depressurizer database or saved AutoCat profiles from Repressurizer's data tools.
4. Preview how imported rules map to current Steam games and cached metadata.
5. Apply a small preset first.
6. Close Steam, create a manual backup, and save.

Imported presets may depend on metadata fields that are missing or represented differently today. Treat the first result as a migration preview, not an automatic source of truth.
