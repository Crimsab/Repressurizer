# Organize a large library

Avoid trying to categorize every game in one pass. Establish a small, stable structure first.

1. Load the library and prepare core Steam metadata.
2. Find `duplicate:true`, `missing:true`, and local-only entries before generating collections.
3. Create a few manual intent collections such as `Playing`, `Backlog`, and `Archive`.
4. Add AutoCat collections for objective facts such as platform, genre, or playtime.
5. Keep generated collections visibly distinct from personal status collections.
6. Preview and save in batches, verifying Steam after each major batch.

This keeps subjective organization under your control while AutoCat handles repeatable metadata rules.
