# Contributing

Repressurizer combines a React and TypeScript frontend with a Tauri and Rust backend. Windows is the supported runtime target, while most static checks and tests can run on Linux.

- [Set up the development environment](setup.md)
- [Understand the architecture](architecture.md)
- [Run the test suites](testing.md)
- [Contribute a translation](../localization.md)
- [Understand the release process](releases.md)

Read the repository [contribution guidelines](https://github.com/Crimsab/Repressurizer/blob/main/CONTRIBUTING.md) before opening a pull request.
